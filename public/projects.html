Senior Design Project- Traffic Generator
Project Manager
Project Page
A traffic generator which generates realistic web traffic against a set of targets for use in Iowa State's Cyber Defense Competition. In the competition, it will obscure traffic coming from the "red" team who try and compromise participants ("blue" team) making the environment much more realistic and nearly impossible to prevent attacks by banning an IP range. In cubersecurity classes, it will allow professors to demonstrate tools like IDS's which can notify users of suspicious attacks, right now there's no good way to do this.

Through this project I have become more proficient in Python, Docker, and RabbitMQ which were our main implementation technologies. Additionally, I learned a lot about networking which was an area I had next to no experience in.

My main contribution to the project was implenting standard web rewquests to our task-running framework. The module reads endpoints from the configuration file and periodically launches a new request towards a randomly selected target.
